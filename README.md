# EXTENSION ITD NEW (EITD)

Расширение для сайта **итд.com**, которое добавляет полезные функции.

---

## Как установить?

### Шаг 1. Установите Tampermonkey
Это расширение, которое запускает пользовательские скрипты.


Нажмите «Установить» и дождитесь окончания.

### Шаг 2. Добавьте скрипт
1. Откройте файл `menu.js` (просто дважды кликните по нему)
2. Скопируйте весь текст (Ctrl+A, затем Ctrl+C)
3. Нажмите на значок Tampermonkey в браузере
4. Выберите «Создать новый скрипт»
5. Удалите всё, что там есть, и вставьте скопированный текст (Ctrl+V)
6. Нажмите «Файл» → «Сохранить» (или Ctrl+S)

### Шаг 3. Зайдите на сайт
Откройте [итд.com](https://xn--d1ah4a.com/) — в левом углу появится пульсирующая бирюзовая кнопка.
