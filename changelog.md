# Краткое предисловие (Что это за проект?)

**EXTENSION ITD NEW (EITD)** — это пользовательский скрипт (userscript) для социальной сети [итд.com](https://xn--d1ah4a.com/), который расширяет функционал сайта через Tampermonkey.

## Основные возможности:
- **Скачивание изображений** — кнопка под постами для сохранения картинок
- **Полная дата регистрации** — отображение точного времени регистрации пользователей в формате `ДД:ММ:ГГГГ ЧЧ:ММ:СС`
- **Кнопка копирования профиля** — копирование JSON-ответа API с данными профиля в буфер обмена
- **Премиум ник** — выделение ника пользователя градиентом и свечением
- **Перетаскиваемое меню** — пульсирующая бирюзовая кнопка с настройками функций
- **Изменение размера окон** — возможность растягивать окна меню и настроек

## Структура:
- `menu.js` — основной файл скрипта (версия 1.3), содержит весь функционал

## Версия 1.3 - Снег и исправления отображения меню

### Добавлено:
- **Функция снега** — анимация падающего снега на странице с настраиваемыми параметрами (количество, скорость, ветер, размер, прозрачность)
- **Настройки снега** — отдельная панель слайдеров в окне настроек для кастомизации эффекта снега

### Исправлено:
- **Баги со скроллингом в меню** — исправлены проблемы с прокруткой содержимого окон меню и настроек
- **Баги с отображением меню** — устранены ошибки рендеринга и позиционирования элементов меню

### Технические детали:
- Функции: `initSnow()`, `stopSnow()`, `_snowDraw()`, `_snowUpdate()`, `_createSnowflakes()`, `_getSnowConfig()`
- Canvas-элемент с `pointer-events: none` и `z-index: 2147483647` для отображения поверх всех элементов
- Анимация через `requestAnimationFrame` с отрисовкой снежинок в виде кругов
- Формула движения: вертикальное падение зависит от `speed` и случайного коэффициента, горизонтальное — синус от угла + ветер
- Параметры сохраняются в localStorage: `itdPlusSnowCount`, `itdPlusSnowSpeed`, `itdPlusSnowWind`, `itdPlusSnowSizeMin`, `itdPlusSnowSizeMax`, `itdPlusSnowOpacity`
- Глобальные функции управления: `window.__setSnowCount()`, `window.__setSnowSpeed()`, `window.__setSnowWind()`, `window.__setSnowSize()`, `window.__setSnowOpacity()` — вызываются при изменении слайдеров
- Автоматическая пересоздание снежинок при изменении количества и размеров
- Остановка: отмена `requestAnimationFrame`, удаление canvas, очистка глобальных функций
- Реакция на resize окна: обновление размеров canvas
