# Краткое предисловие (Что это за проект?)

**EXTENSION ITD NEW (EITD)** — это пользовательский скрипт (userscript) для социальной сети [итд.com](https://xn--d1ah4a.com/), который расширяет функционал сайта через Tampermonkey.

## Основные возможности:
- **Скачивание изображений** — кнопка под постами для сохранения картинок 
- **Полная дата регистрации** — отображение точного времени регистрации пользователей в формате `ДД:ММ:ГГГГ ЧЧ:ММ:СС`
- **Перетаскиваемое меню** — пульсирующая бирюзовая кнопка с настройками функций

## Структура:
- `menu.js` — основной файл скрипта (версия 1.2.2), содержит весь функционал

# Changelog

## Версия 1.2.2 - Полная дата регистрации

### Что добавлено:
- Функция отображения полной даты регистрации пользователей
- Точное время регистрации в формате `ДД:ММ:ГГГГ ЧЧ:ММ:СС`
- Переключатель «Полная дата регистрации» в меню настроек
- Автоматический перехват API-запросов для получения даты регистрации
- Сохранение настройки в localStorage

### Возможности:
- Отображение даты и времени регистрации под постами
- Поддержка различных форматов API-ответов (`createdAt`, `author.createdAt`, `profile.createdAt`)
- Включение/выключение функции через меню настроек
- Автоматическое применение при загрузке страницы

### Технические детали:
- Перехват `fetch`-запросов к API пользователей
- Форматирование даты через `formatRegDate()`
- Инъекция даты в элементы `.at4eWYfl` с иконкой
- Функции: `window.setFullDateEnabled()`, `window.getRegDate()`, `window.forceInjectRegDate()`

## Версия 1.2 - Меню с настройками и слайдером

### Что добавлено:
- Пульсирующая бирюзовая кнопка меню (перетаскиваемая)
- Перетаскиваемое окно меню с настройками
- Слайдер-переключатель "Кнопка скачивания постов"
- Отдельное окно настроек (открывается по иконке шестерёнки)
- Сохранение позиций окон в localStorage

### Возможности:
- Включение/выключение кнопок скачивания через меню
- Перетаскивание окон за хедер
- Сохранение позиций между сессиями
- Анимация пульсации кнопки меню
- Ховер-эффекты на всех кнопках

### Изменения:
- Функционал скачивания картинок перенесён из `main.js` в `menu.js`
- Кнопки скачивания появляются только при включённом слайдере
- При выключении слайдера кнопки скачивания удаляются из постов
- `main.js` Удалён.

### Технические детали:
- Добавлен `@grant GM_xmlhttpRequest` для скачивания
- Используется localStorage для хранения настроек
- CSS-анимации для пульсации и слайдера

## Версия 1.0 - Добавлена кнопка скачивания картинок `main.js`

### Что добавлено:
- Кнопка скачивания картинок под каждым постом с изображениями
- Расположение: в панели действий, сразу после кнопки "Репост"
- Иконка: кастомная SVG из файла `download_ico.svg`

### Возможности:
- Скачивание одной картинки по клику
- Пакетное скачивание нескольких картинок (по очереди с задержкой 200мс)
- Обход CORS ограничений через `GM_xmlhttpRequest`
- Автоматическое имя файла: `itd_post_{postId}_{timestamp}.jpg`
- Реактивное обновление при динамической подгрузке постов (лента, скролл)

### Технические детали:
- Платформа: Tampermonkey
- Домен: https://итд.com/*
- Использует MutationObserver для отслеживания новых постов
- Встраивание SVG-иконки напрямую в код
